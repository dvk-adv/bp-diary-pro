📖 О проекте
Дневник АД • Pro — это лёгкое веб-приложение для ежедневного контроля артериального давления и пульса. 

Создано с заботой о людях, которым важно следить за здоровьем сердечно-сосудистой системы, но которые 
не хотят разбираться в сложных медицинских сервисах.

💡 Знаете ли вы? По данным ВОЗ, более 1,28 миллиарда взрослых в мире страдают гипертонией. Регулярный самоконтроль АД снижает риск инсульта на 40% и инфаркта на 25%.

Простота использования
Три шага для начала работы:
Откройте файл bp-diary.pro.html в любом современном браузере
Заполните профиль — имя, пол, год рождения
Добавьте первое измерение — готово!
Никакой установки. Никакой регистрации. Никаких платежей.

🛠️ Технологии
HTML5 / CSS3 / JavaScript — чистый код без фреймворков
Chart.js — визуализация данных
docx.js — генерация отчётов в формате Word
LocalStorage — хранение данных в браузере
Web Speech API — голосовой ввод

📋 Системные требования
Любой современный браузер (Chrome, Firefox, Safari, Edge)
Поддержка JavaScript
Около 2 МБ свободного места
